ref 获取 dom 元素

首先在改组件中的标签中使用 ref=”name”

然后在实例化中使用 this.$refs={…}

所以 this.$refs.name=DOM 元素

一般使用

  1. 获取基本元素
  2. 获取子组件,直接调用其中的方法

操作 DOM 注意

在操作 更改过的 DOM 时,因为 vue 的生命周期,使用 this.$nextTick( funtion( ){ } )

Keep-alive

在使用子组件时,不止可以用

也可以指定要显示的数据

使用的 name 为声明组件的 name

更改显示时 ,会被销毁,所以使用 keep-alive 保证不会被回收

具有两个参数:

  1. include:include="name1,name2"
  2. exclude:exclude="name1,name2"

slot 插槽

父组件使用…..指定内容,子类使用 ,表示父类定义内容的存放点在 slot 中不指明 name=” xxx”,那么默认为 default,父类不指明 v-slot:name,默认为 default
